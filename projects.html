Dress Code

Project Date & Duration: March 2025 – April 2025 (1 month)

Key Skills: UI design, Java programming in Eclipse, digital illustration, interaction design, state-based programming, and simulation logic

Dress Code is a cozy, interactive dress-up simulation that invites users to explore creativity through fashion and storytelling. Featuring hand-drawn visuals and intuitive drag-and-drop mechanics, the game lets players mix and match outfits while discovering playful surprises. Designed with a state-based system, it offers a seamless and charming user experience that sparks imagination and nostalgia.

Princess Quest

Project Date & Duration: July 2024 – August 2024 (2 months)

Key Skills: 2D game design, p5.js programming, pixel art, interaction design, platformer mechanics, and narrative development

Princess Quest is a whimsical 2D pixel platformer that blends classic game charm with playful storytelling. Players guide a brave princess through fairy tale lands, leaping across platforms and avoiding mischievous ghosts in search of her lost prince. Built using p5.js, the game combines simple controls, retro visuals, and a light-hearted narrative to create an engaging, nostalgic adventure full of surprises and charm.
